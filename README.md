# PhotoBooth

An online Photo Gallery website where users can login and upload photos. 
With this application, users can safely save and organize their photos and access them from anywhere. 

**Tech Stack:**
- Node.js
- React
- Firebase
- OAuth
- Semantic UI
- Framer Motion
